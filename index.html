Projects

Smart Irrigation System

Es kontrolliert dauerhaft die Bodenfeuchtigkeit-Werte und bewässert wenn nötig. Zeigt den Feuchtigkeitswert und die IP-Adresse auf einem Display an. Speichert den Wert alle 30 Minuten in eine Datenbank auf meinem Heim-Server. Der Microcontroller hostet eine Website, auf welcher der Wert angezeigt wird. Der Heim-Server hostet eine Website, auf der ein Graph mit der Wertentwicklung angezeigt wird. Es gibt auch eine Version, die den Arduino uno nutzt. Diese unterstützt aber kein Daten-Tracking
Verwendete Technologie:
C++, ESP 32 Controller, Mysql Datenbank, PHP

DirectX 11 Game Engine

This is by far my largest project—a custom 3D game engine built directly on Microsoft’s DirectX API. Initially developed with DirectX 9 and later upgraded to DirectX 11, this engine emphasizes low-level programming, providing direct control over GPU resources and enabling precise hardware interaction. The project highlights the principles of efficient resource management and performance optimization, crucial in both embedded systems and high-performance graphics.

Technologies Used:
C++, DirectX 11 (formerly DirectX 9), Nvidia PhysX

Vulkan Graphics Engine

This project is a highly modular and cross-platform graphics engine built with Vulkan and modern C++ techniques. Designed specifically for voxel graphics, it aims to leverage Vulkan's cutting-edge capabilities for optimal performance and flexibility. The engine is still in its early stages, being developed from the ground up to ensure modernity and modularity. This project is a learning journey into Vulkan and the latest C++20 features, driven by a desire to stay at the forefront of graphics technology.

Technologies Used:
C++, Vulkan, CMake

Embedded Machine Learning System

This project explores the challenge of deploying AI models on an ESP32 microcontroller, a device with extremely limited resources and no file system. The goal was to transfer a machine learning model from a Jupyter Notebook environment to a microcontroller, ensuring it runs efficiently despite the hardware constraints. By leveraging C++ for embedded programming and Python for model development, I successfully integrated AI capabilities into a low-power device. This project demonstrates the feasibility of running sophisticated AI algorithms on minimal hardware.

Technologies Used:
C++, ESP32 Controller, Python, Keras, TensorFlow

Machine Learning and AI

This project is a personal collection of machine learning experiments and implementations, focused on deepening my understanding of AI. Key highlights include developing GANs, diffusion models, CNNs, and NLP models using the Shakespearean dataset. Additionally, I implemented the "Attention Is All You Need" model, the foundational architecture behind technologies like ChatGPT. The primary motivation was to learn the underlying mathematics and practical implementation of machine learning models from scratch.

Technologies Used:
Python, Keras, TensorFlow, scikit-learn

2D Game Development Framework

This project is a custom-built 2D game development framework designed to streamline the creation of 2D games. It includes support for the Tiled Editor format, a collision detection system, A* pathfinding, and a UI system for HUDs and main menus. The framework was developed to enhance my understanding of the A* algorithm and improve my C++ programming skills. Building this framework allowed me to dive deeper into game development mechanics while creating a versatile tool for 2D game projects.

Technologies Used:
C++, SDL2
